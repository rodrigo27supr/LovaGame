# LovaGame: Mi Primer Proyecto de Java
![Estado del Bot](https://github.com/rodrigo27supr/LovaGame/actions/workflows/ejecutar.yml/badge.svg)

Hola, soy **Rodrigo**, un estudiante de desarrollo de aplicaciones web.

Este repositorio es el inicio de mi portafolio profesional. **LovaGame**.

## La historia detrás del código

Mi objetivo con este proyecto era salir de la teoría y construir algo útil y funcional.

Para lograrlo, he contado con la **ayuda de la Inteligencia Artificial Gemini que ha actuado como mi mentora**, pero me gustaría aclarar que **no he copiado y pegado código sin pararme a pensar**

He seguido los siguientes pasos para que fuera un proyecto lo más didáctico posible:

- El código ha sido escrito por mí, paso a paso.

- He comentado varias líneas del código con ayuda de la IA para explicarme el porqué y para asegurarme de entender la lógica detrás de cada bucle, cada conexión y cada archivo.

- No quería que se quedara en mi ordenador, así que aprendí a configurar GitHub Actions para tenerlo en la nube.

##  ¿Qué hace LovaGame?

Es un bot desarrollado en **Java** que:
1.  Rastrea internet cada hora buscando ofertas de videojuegos 100% gratuitos.
2.  Filtra los resultados para no repetir avisos (usando un historial).
3.  Envía una alerta a mi canal de difusión en Telegram.

## Tecnologías que he usado en mi proyecto
En este proyecto he puesto en práctica:
* **Java 17:** Manejo de objetos, listas y control de flujo.
* **APIs REST:** Consumo de datos externos en formato JSON.
* **Git & GitHub:** Control de versiones y manejo de repositorios.
* **GitHub Actions:** Automatización de tareas en la nube.
* **Seguridad:** Gestión segura de tokens y contraseñas (Secrets).
